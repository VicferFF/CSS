Introducción a CSS


1. ¿Qué es CSS y para qué sirve?

De forma resumida, la función de CSS es hacer que nuestra página web sea mucho más estética, ya que es este el que nos permite modificar el color de los elementos (letra, fondo, tablas, etc), el tamaño, formato, posición, etc de la letra y de las imagenes que insertemos en el código HTML, entre otras detalles que hacen que nuestra página web se vea mejor y más personalizada.


2. Formas de aplicar CSS en HTML

CSS en línea (inline): Esta forma utiliza el atributo HTML "style", utiliza atributos HTML definidos como globales (aplicables a todos los elementos). Este tipo de declaración se utiliza de forma individual a cada registro HTML del código de la página. Aquí un ejemplo:

Ejemplo de CSS inline poniendo las letras rojas.


CSS en el encabezado (internal): Esta forma utiliza el atributo "style" en la sección "head" del documento HTML. Este estilo CSS es un método que afecta a toda una página, por lo que es recomendable si nuestra web va a tener pocas paginas. El ejemplo lo encontramos dentro de la etiqueta head del código esta misma pagina, haciendo que el fondo sea gris.


CSS externo: Con esta forma se enlazan las páginas web a un archivo .css externo. Se le considera a este tipo de CSS como el más eficiente, ya que puede estilizar un sitio web grande, pudiendo editar un archivo .css y cambiar todo un sitio web de una sola vez. El ejemplo tambien lo encontramos dentro de la etiqueta head del código esta misma pagina, haciendo que todas las letras de cada "h2" sean amarillas
