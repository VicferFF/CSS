Introducción a CSS


1. ¿Qué es CSS y para qué sirve?

De forma resumida, la función de CSS es hacer que nuestra página web sea mucho más estética, ya que es este el que nos permite modificar el color de los elementos (letra, fondo, tablas, etc), el tamaño, formato, posición, etc de la letra y de las imagenes que insertemos en el código HTML, entre otras detalles que hacen que nuestra página web se vea mejor y más personalizada


2. Formas de aplicar CSS en HTML

CSS en línea (inline): Esta forma utiliza el atributo HTML "style", utiliza atributos HTML definidos como globales (aplicables a todos los elementos) La declaración tiene lugar a nivel de las etiquetas individuales contenidas en la página, por lo que se habla de hojas de estilo en línea.
