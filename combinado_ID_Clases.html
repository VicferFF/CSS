Desafíos adicionales

En HTML, el id está pensado para identificar de manera única a un elemento dentro de la página. Por lo que cada id debe ser único, ya que su propósito es brindar un identificador especifico a un solo elemneto.

Cuando un elemento tiene una clase y un id, es el id el que tiene mayor peso. En este ejemplo, en el párrafo con el id "especial" y la clase "contenido", el color de fondo del id teiene prioridad ya que se ve que primero tiene un color de fondo gris con la clase "contenido" y luego cambia a verde por el el id "especial" .
