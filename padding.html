CSS margin

El margin es el espacio exterior que rodea un elemento HTML, enotras palabras, el espacio entre el elemento y los otros elementos que lo rodean. Utiliza los siguietes comandos de código: margin-top, margin-right, margin-bottom, margin-left.


CSS padding

El padding es el espacio interior entre el contenido de un elemento HTML y el borde del mismo elemento. Los comandos de código son similar a los del margen: padding-top, padding-right, padding-bottom, y padding-left.


CSS border

El border es la línea que rodea un elemento HTML, se encuentra entre el padding y el margin. Se le puede definir el grosor, el estilo (sólido, punteado, doble, etc.), y el color del borde.


CSS width and height

Width y height especifican el ancho y la altura de un elemento HTML, sirven para definir el tamaño del área de contenido del elemento. Puedes usar valores absolutos como "píxeles" o valores relativos como "porcentaje", dependiendo de cómo quieras que el elemento se ajuste al contenedor.
