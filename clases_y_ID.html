Cuándo es más adecuado usar una clase o un identificador.

Las Clases se usan cuando quieres aplicar un mismo estilo a múltiples elementos y los Identificadores se usan cuando necesitas un estilo único para un solo elemento.


Cómo se utilizan en el HTML y cómo se referencian en el CSS.

En HTML las clases se añaden con el atributo "class", por ejemplo:

...

y los identificadores se añaden con el atributo id, por ejemplo:

...

.

En CSS, las clases se refieren con un punto (.), por ejemplo: .texto { color: blue; } y los identificadores se refieren con este símbolo (#), por ejemplo: #titulo { color: pink; }.


Las principales diferencias en cuanto a su uso y especificidad.

La especificidad de un id es mayor que la de una class, por lo que, en caso de conflicto de estilos, los estilos definidos por un id tienen prioridad sobre los definidos por una class. Las clases se usan para agrupar elementos que comparten características visuales, mientras que los identificadores se reservan para dar un estilo único a un elemento específico.
